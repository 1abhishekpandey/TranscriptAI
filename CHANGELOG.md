# Changelog

All notable changes to TranscriptAI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-09

### Added
- YouTube transcript download using InnerTube API integration
- Self-contained YouTube subtitle downloader extension module
- Clean and intuitive subtitle viewer with scrollable interface
- Export functionality:
  - Copy transcripts to clipboard
  - Share directly to ChatGPT/Claude apps
  - Export as text file
- Clean Architecture implementation with MVVM pattern
- Material3 design system
- Comprehensive documentation:
  - Architecture guidelines
  - Coding standards
  - Build commands reference
  - YouTube InnerTube API guide
- Recency-based sorting for prompt selection with persistence
- Text sharing options for seamless AI platform integration

### Technical Details
- Min SDK: 25 (Android 7.1)
- Target SDK: 36 (Android 15)
- 100% Kotlin with Jetpack Compose
- Kotlin Coroutines for async operations
- Edge-to-edge display support
